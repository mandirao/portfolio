# spec: design refresh v1

## What this is
A first-pass visual and copy refresh of the existing scaffolded 
site to match the design direction and voice defined in CLAUDE.md.

## What to change

### Copy
- Rewrite all placeholder copy using the voice guidelines in 
  CLAUDE.md: direct, specific, outcomes over outputs, no buzzwords
- Hero headline should lead with a tension or pattern, not a title
- Do not invent metrics or stories — use only what's in CLAUDE.md 
  under "What I bring that's unusual"

### Design
- Apply stone color palette with terracotta as the single accent
- Increase typographic hierarchy — size and weight should carry 
  structure before color does
- Add whitespace generously — pacing over density
- Remove any template-feeling elements (generic card layouts, 
  placeholder grid structures)

## What NOT to change
- File structure (no new files unless necessary)
- The Nav component
- Any routing or Next.js configuration

## Definition of done
- Looks intentional, not scaffolded
- Copy sounds like one person wrote it
- No placeholder text remaining on home page
- Committed and pushed to GitHub
